Thinking Mode Under Fixed Retrieval

The hard-20 benchmark was rerun with fixed Qwen embeddings to compare Qwen3.5 model size and thinking mode under the same retrieved evidence.

evaluationqwenthinking

At a glance

  • Research question: Does thinking mode improve answer quality when retrieval evidence is fixed?
  • What changed: We rebuilt the hard-20 run with Qwen/Qwen3-Embedding-0.6B, regenerated gold answers from the fixed Top-5 evidence, and tested Qwen3.5 0.8B, 2B, and 4B in non-thinking and thinking modes.
  • Main result: Thinking mode did not consistently improve this RAG task. The 4B non-thinking run remained the strongest Qwen setting on faithfulness and answer correctness.
  • Current limitation: Only 8 of 20 questions were marked answerable under the fixed Top-5 evidence, so the gold labels still need manual audit.

Method

This run used one retrieval cache for all answer models.

  • Retrieval index: structural_fallback_v1.
  • Embedding model: Qwen/Qwen3-Embedding-0.6B.
  • Answer models: Qwen/Qwen3.5-0.8B, Qwen/Qwen3.5-2B, and Qwen/Qwen3.5-4B.
  • Modes: non-thinking and thinking.
  • Gold answers: regenerated from the fixed Top-5 evidence with openai/gpt-5-mini.
  • RAGAS judge: openai/gpt-5-mini.
  • Evaluator embedding model: Qwen/Qwen3-Embedding-0.6B.

Reasoning diagnostics were also generated for the 0.8B thinking and 4B thinking runs. The diagnostic splits saved reasoning text by newline, embeds each step, projects the vectors with PCA, and clusters them.

Results

The model-level result does not show a general benefit from thinking mode.

  • Qwen3.5 0.8B non-thinking: faithfulness 0.601, relevancy 0.643, correctness 0.268, semantic similarity 0.545.
  • Qwen3.5 0.8B thinking: faithfulness 0.585, relevancy 0.580, correctness 0.358, semantic similarity 0.563.
  • Qwen3.5 2B non-thinking: faithfulness 0.734, relevancy 0.617, correctness 0.383, semantic similarity 0.559.
  • Qwen3.5 2B thinking: faithfulness 0.699, relevancy 0.435, correctness 0.360, semantic similarity 0.580.
  • Qwen3.5 4B non-thinking: faithfulness 0.823, relevancy 0.641, correctness 0.420, semantic similarity 0.563.
  • Qwen3.5 4B thinking: faithfulness 0.817, relevancy 0.560, correctness 0.387, semantic similarity 0.590.

The 4B model is stronger than 0.8B on faithfulness and answer correctness. Thinking mode slightly improves semantic similarity in this run, but it lowers answer relevancy.

Interpretation

The useful claim is limited. Thinking mode is not automatically better for retrieval-grounded student Q&A. It can produce more internal text and slightly higher semantic similarity, but it can also reduce answer relevancy.

For the thesis, this supports a fixed-retrieval evaluation design. Model size and thinking mode should be measured under the same evidence rather than assumed to improve quality.

Next step

  • Manually audit the regenerated gold labels, especially the questions marked insufficient evidence.
  • Compare Qwen3.5 4B non-thinking against smaller prompt-tuned settings before trying fine-tuning.
  • Treat the reasoning trajectory plots as exploratory diagnostics, not as primary answer-quality evidence.

The key result is conservative: 4B does not have a larger trajectory diameter in this run. Its mean and global diameters are slightly smaller. The visible improvement is instead that more of its reasoning steps belong to high-correctness answers and fewer belong to low-correctness answers.
