System30 Model and Mode Sweep

Remote pgvector/BGE runtime was evaluated on 30 regression questions across six small models and two answer modes.

evaluationsystem30agentic ragmodel sweep

Purpose

This update records a system-level regression run after moving the runtime to the remote ICU Wiki server. The goal was to test the current retrieval and answer path on questions that are closer to real student use than the earlier benchmark.

Evaluation setup

  • The benchmark contains 30 questions: 20 inherited hard cases and 10 new regression cases.
  • The new cases focus on English mandatory-course questions, ChatGPT and senior-thesis policy boundaries, teacher-name lookup in Japanese and Chinese, and course-registration completion procedures.
  • The runtime used PostgreSQL hybrid retrieval, local BGE-M3 embeddings, local BGE reranking, Janome tokenization, entity-aware retrieval, source-authority scoring, and a reranker candidate limit of 40.
  • The sweep tested Qwen3.5 0.8B, 2B, 4B, 9B, Gemma 4 E2B, and Gemma 4 E4B.
  • Each model was run in single-shot mode and agentic mode, giving 360 total answers.

Main result

  • Qwen3.5 9B was the strongest setting in this run.
  • Qwen3.5 9B agentic reached the best average human-quality score and the best clean-pass rate.
  • Qwen3.5 9B single-shot was close in quality and much faster on average.
  • Gemma 4 E2B and E4B were competitive on readability and citation presentation, but more often missed key conditions.
  • Qwen3.5 0.8B was below the current practical threshold for a student-facing answer path.

Interpretation

Agentic mode is useful when the question needs query planning, evidence judgment, or a second retrieval attempt. The cost is latency. For the current system, the clearest tradeoff is between Qwen3.5 9B single-shot as the faster strong baseline and Qwen3.5 9B agentic as the higher-quality diagnostic path.

The remaining weak cases are not only model-size problems. Several failures come from policy boundary questions where the retrieved text is partial, old, or too indirect. These cases need stricter abstention and better evidence selection, not just a larger model.
